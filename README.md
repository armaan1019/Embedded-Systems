# Embedded-Systems
SCU coursework covering embedded systems labs. Programming in Assembly ARM 64 bit architecture along with C.

This course covers how to program in Assembly and use the STM32 MCU to flash the programs. The topics of this course are as follows:

1) Architecture: CPU, registers, buses, memory, I/O interfaces

2) Assembly: Addresses, arithmetic and logic operations, IT branching and for loops, stack operations, parameter passing

3) C: Pointers, memory management, stack frames

The labs cover the following:

Lab 1: Basic operations in Assembly 

Lab 2: Writing a C program to convert to octal/hexadecimal and 2's compliment

Lab 3: Writing C methods in Assembly

Lab 4: Optimization worksheet

Lab 5: Linear interpolation: writing a C function in Assembly

Lab 6: Sliding puzzle: replacing C code with Assembly

Lab 7: Autonomous Sudoku

Lab 8: Resistor color codes
